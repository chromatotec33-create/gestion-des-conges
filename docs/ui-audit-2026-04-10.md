# Audit UX/UI — 10 avril 2026

## Constats principaux

1. **Navigation trop orientée structure interne**
   - libellés techniques pour des publics non experts.
   - manque de distinction claire salarié / manager / RH.

2. **Complexité multi-société trop visible**
   - le switch société est affiché en permanence, y compris pour usage quotidien salarié.

3. **Guidage insuffisant**
   - peu d'entrées "Je veux..." pour aider un utilisateur peu à l'aise.
   - parcours assistés pas mis en avant dans le header.

4. **Manque de rassurance opérationnelle**
   - absence d'encarts de support rapide (aide, FAQ, actions fréquentes) dans le layout principal.

## Objectifs de la refonte layout (itération actuelle)

- Simplifier et clarifier la navigation principale.
- Masquer la complexité multi-société en usage standard.
- Introduire des actions guidées orientées métier terrain.
- Renforcer lisibilité et accessibilité de la structure globale.

## Périmètre de cette itération

- Refonte du layout principal (sidebar, header, navigation, contexte société).
- Mise en place d'une navigation par profils d'usage (salarié / manager / RH).
- Intégration d'actions rapides "Je veux..." et accès aide.
